# Employee Management System (OOP Wrapper Project)

# 📌 Project Overview

The Employee Management System is a Python-based console application developed using Object-Oriented Programming (OOP) concepts. The project demonstrates how real-world employee data can be managed efficiently through classes, inheritance, encapsulation, method overloading, method overriding, constructors, destructors, and other OOP features.

The system allows users to create and manage different types of employees such as Managers and Developers, while maintaining their details like employee ID, name, age, salary, department, and programming language.

This project is designed for educational purposes to help students understand and implement core OOP concepts in Python.

# 🎯 Objectives

The main objectives of this project are:

◾Implement Object-Oriented Programming concepts in Python.

◾Create a base Employee class and specialized derived classes.

◾Demonstrate inheritance and method overriding.

◾Implement encapsulation using private attributes.

◾Use getter and setter methods for secure data access.

◾Demonstrate constructor overloading using default arguments.

◾Utilize super() to access parent class functionality.

◾Use issubclass() to verify class relationships.

◾Create a menu-driven user interface for user interaction.
